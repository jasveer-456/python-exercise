# Python Practice task
---
Take a list of 5 numbers as input from the user.
# List Operations
---
* Print the original list.
* Find and display the sum, maximum, and minimum.
* Append a new number to the list.
* Insert a number at a specific index (2)
  # Sample Run

  # Input
```
  x=[1,2,3,5,9]
print("original array : ",x)
x.append(28)
print("append : ",x)
x.insert(2,15)
print(x)
print("Sum : ",sum(x))
print(x)
print("min : ",min(x))
print("max : ",max(x))
```
# Output
```
original array :  [1, 2, 3, 5, 9]
append :  [1, 2, 3, 5, 9, 28]
[1, 2, 15, 3, 5, 9, 28]
Sum :  63
[1, 2, 15, 3, 5, 9, 28]
min :  1
max :  28
```
